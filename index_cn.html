Browsing Insights

关键词: Chrome 扩展, Chrome API, HTML, CSS, JavaScript

概览

“Browsing Insights”为希望细化查询，追溯和分析浏览历史的用户提供了实用功能。它包含高级搜索、自定义过滤器， 以及访问趋势等功能，可为用户提供不同网站访问量的图表，更好地反映和追溯用户的网上浏览活动。 可从Chrome Web Store 安装。

项目相关

默认的浏览器历史工具在功能和灵活性方面有所欠缺。像Google Chrome或Microsoft Edge，它们的浏览历史界面只简单地按照时间倒序展示用户访问过的网站列表。 缺乏灵活性和易于用户总结，回看反思的功能。看到这一问题，本研究旨在开发一个Google Chrome浏览器的扩展程序，该程序具有搜索、排序、按日期、时间段或站点域名自定义追溯浏览历史的功能，并可提供不同网站访问量的图表。

在技术方面，该工具使用Chrome历史API获取浏览数据， 融入了用户友好型功能，如无限滚动、防抖搜索、键入信息建议列表 、防错机制等，并在程序首页提供了基础使用指南。此外，在网站访问数据可视化中，用户可根据自己的喜好选择展示堆叠或非堆叠的比较条形图，以便更直观地理解自己的浏览行踪。 同时，用户可以自行选择需要追踪的网站，或者选择将某些网站排除在追踪之外。

Netflix影视作品分析

关键词: 网络爬虫, 数据可视化, selenium, matplotlib, 情感分析

研究目标

这个项目意在爬取、分析和可视化Netflix从2012年到2023年发布的影视作品数据，并揭示其影视发行量趋势、种类的多样性、不同流派 的受欢迎程度和影迷评分 。以此了解Netflix的内容创造策略是如何与观众偏好相匹配，并逐步扩大影响力的。

研究发现

Netflix原创作品的数量从2012年到2020年稳步上升，在2022年达到高峰，之后在2023年略有下降。从2017年到2020年，各种流派的制作逐年都有所增长，其中剧情和 喜剧类作品增长最为明显。战争、脱口秀、西部和音乐剧类型的作品从2017年到2023年发行数量一直相对稳定。在2022年发行量达到高峰之后，所有类型在2023年都有所下 降，除了历史和惊悚类型反而有所增加。

在节目的内容分级中，最常见的是MA15+/M（适合成年观众），且各评级的IMDb分数均值没有显著差异。不同流派的IMDb分数均值差异较大，其中历史和战争类作评分最高， 而恐怖和爱情类作品评分最低。

经过分析，可以发现Netflix发行量最多的节目类型与每年在IMDb上投票量（关注度）最高的节目类型高度一致，主要包括剧情、喜剧、纪录片和犯罪。然而，值得注意的是， 这些节目的IMDb评分相较2016年前呈现出下降的趋势。与这些作品相关的关键词主要包括“世界”、“新”、“关系”、“父亲”和“朋友”等。

另外，Netflix每年发布的IMDb评分 最高的前10个节目主要归属于剧情、喜剧、犯罪和动画等类型。通过对这些节目的描述进行关键词分析，我们发现最常见的关键词包括“家庭”、“孩子”、“主角”和“朋友”。

Twist Con

关键词: Unity 3D, VR 游戏, Arduino

概览

该项目的目标是设计一款增强触感体验的VR手柄。传统VR手柄虽然满足常见的交互方式，但在为特定场景提供逼真的触感反馈 方面存在不足。而Twist-Con 就是专为模拟各种扭动动作和场景的触感而设计的VR手柄。

项目相关

控制器的主体部分是一个带内置编码器的小电机，该电机利用PID等控制逻辑来实现扭转阻力的触觉反馈，并通过Arduino和UNO板进行编程。电机的固定外壳 由激光切割的中纤板拼接构成，此外，该外壳还连接了了臂架，以增加灵活性和舒适性。电机轴上配置了三种不同的3D打印旋钮，使用户能够与各种形状的物体进行扭转交互。

这款在Unity中开发的VR游戏，设有三个独特的场景，旨在让用户沉浸在以下触感体验中：操纵摩托车把手、旋转胡椒研磨器，以及上紧玩具机器人的发条。
